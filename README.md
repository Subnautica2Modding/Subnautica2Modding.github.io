# Subnautica 2 Modding

Community documentation for modding Subnautica 2, available at **https://subnautica2modding.github.io/**.

## About

This site provides guides, references, and tutorials for creating mods for Subnautica 2. Whether you're just getting started or looking for specific technical references, you'll find everything you need here.

## Content

- **Beginner's Guide to Modding Subnautica 2** - a step-by-step introduction to creating your first mod

More guides and references are added regularly by the community.
