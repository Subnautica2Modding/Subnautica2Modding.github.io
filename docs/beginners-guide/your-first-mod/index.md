# Your first Subnautica 2 mod

This is where it starts to get fun! Let create a mod that will give the player unlimited oxygen and also prevent starvation. It's actually pretty straightforward to code, but the challenge with modding typically comes not from writing the code, but figuring out how to get the code hooked into the right place, and affecting the right objects and values in the game.

If you're stuck, or just need a bit of a nudge in the right direction, you can always refer to the working source code in my GitHub repository:

[BeginnersGuideCheatMod](https://github.com/mroshaw/Subnautica2Mods/tree/main/mods/BeginnersGuideCheatMod)
