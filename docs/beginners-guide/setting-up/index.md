# Setting up for development

Before we can do anything, we need to set up our development environment. I use a Windows machine, so the software and installation processes I'm documenting assume Windows as a target operating system. I'm sure there are ways and means to get things working on MacOS or Linux, but that's outside the scope of this tutorial.

If you've got a PC beefy enough to run the games, you'll be just fine to install and configure a modding environment. So let's download and install what we need.
